## 🚀 Overview

SPARE SYNC is an AI-powered motorcycle spare parts compatibility platform that helps riders, mechanics, workshops, and spare-parts dealers determine whether a specific bike part can be used across different motorcycle models.

The system combines:

* AI Compatibility Engine
* Intelligent Parts Matching
* Voice Assistant
* Natural Language Chatbot
* Real-Time Compatibility Analysis
* Parts Marketplace
* Seller-Buyer Chat Interface

All features run directly in the browser using pure HTML, CSS, and JavaScript without requiring a backend server.

## ⚙️ Compatibility Engine

The compatibility engine evaluates parts using technical specifications such as:

* Engine displacement (CC)
* Cooling type
* Chain specification
* Wheel size
* Brake system
* ABS configuration
* Fork type
* ECU compatibility

A compatibility score is generated:

* 80–100% → Fully Compatible
* 50–79% → Partially Compatible
* 0–49% → Not Compatible
